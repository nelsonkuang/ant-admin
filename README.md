# ant-admin
基于 antd 的后台管理平台, 在react中玩转D3.js   

[在线 Demo](http://www.a4z.cn/pui/ant-admin.html#/vertical-bp-chart)    

## 如何使用?

此项目使用`Create React App`初始化的   
npm下可用的命令   

``` 
npm start   
npm test   
npm run build   
npm run eject   

```

yarn下可用的命令   

``` 
yarn start   
yarn test   
yarn build   
yarn eject   

```
